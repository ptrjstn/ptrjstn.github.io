# ptrjstn.de

Persönliche Website von Peter Justen mit Landingpage, Work und About.

Die Landingpage erzeugt serverseitig über `api/neologism.js` bei jedem Besuch
ein neues deutsches Wort. Dafür muss `OPENAI_API_KEY` als Vercel Environment
Variable gesetzt sein. Der Schlüssel wird ausschließlich serverseitig verwendet.
